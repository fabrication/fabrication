InheritanceExpression flash.utils.Proxy
ImplementsIDisposable

Expression is a dynamic class used to build the path to a UI component. wrt to its DisplayObject hierarchy. Expressions can be attached to a root component resolver which specifies the base expression. Expressions can be named, descendant, regular expressions or descendant with regular expressions.
  • named : .componentName
  • descendant : ..componentName
  • regular expression : .re(/componentName/)
  • descendant with regular expression : ..rex(/componentName/)

The implementation uses regular expressions for all types of expressions. The named and descendant expressions are converted into regular expressions internally.
