Classpublic class FlexModuleLoader
InheritanceFlexModuleLoader mx.modules.ModuleLoader
ImplementsIFabrication

FlexModuleLoader is the concrete application fabrication for loading flex modules. It implements IFabrication to allow configuration of the module prior to it has completed loading.
