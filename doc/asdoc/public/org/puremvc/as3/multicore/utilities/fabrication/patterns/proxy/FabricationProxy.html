getNotificationName()method 
public function getNotificationName(note:String):String

Returns the full name for the specified notification. If the name of the Proxy is the same as Proxy.NAME then no prefix is used else the name of the proxy is prefixed to the notification.

Parameters
note:String

Returns

Example
For the Proxy SampleProxy with the name MySampleProxy, the notification Changed becomes, MySampleProxy/Changed
