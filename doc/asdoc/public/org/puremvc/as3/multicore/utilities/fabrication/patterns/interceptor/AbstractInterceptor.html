InheritanceAbstractInterceptor SimpleFabricationCommand org.puremvc.as3.multicore.patterns.command.SimpleCommand
ImplementsIInterceptor

The AbstractInterceptor is the base class for creating interceptors. It provides most of the implementation of the IInterceptor interface. Subclasses only need to implement the intercept method.
