Classpublic class FabricationFacade
InheritanceFabricationFacade org.puremvc.as3.multicore.patterns.facade.Facade
ImplementsIDisposable

FabricationFacade is a concrete PureMVC facade implementation that provides the typical bootstrap process of a PureMVC application. It implements a startup method to kickstart the PureMVC apparatus with a STARTUP notification.

startup()method 
public function startup(startupCommand:Class, application:Object):void

Starts the PureMVC apparatus after registering and triggering a STARTUP notification. A BOOTSTRAP notification is also sent to allow the startup commands to be split into 2 parts, a creation phase and a configuration phase.

Parameters
startupCommand:Class — The startup command class for this application
 
application:Object — The main application reference
