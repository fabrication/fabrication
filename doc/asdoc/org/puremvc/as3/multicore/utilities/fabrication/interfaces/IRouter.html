connect()method
public function connect(cable:IRouterCable):void

Connects the router cable to the router. The router cable describes the input and output pipe names of the module. These are used to connect to the junction.

Parameters
cable:IRouterCable — The router cable for the module to connect
disconnect()method 
public function disconnect(cable:IRouterCable):void

Disconnects the router cable from the router. The router cable need not be the same object but must have the same input and output names.
